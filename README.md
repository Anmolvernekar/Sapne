# Sapne
Sapne-Wishlist management system is a tool that would help user to organize their lifetime travel goals and set priorities for them based on how soon they would like to achieve them. This is a user-friendly tool for managing bucket list, in this the user can create their travel wish list and fulfill their dreams, for example the list of places they have not yet visited but wish to visit.
